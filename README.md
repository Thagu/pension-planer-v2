# 🧮 Pension Planer v2

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
▼ ![Status](https://img.shields.io/badge/Status-In_Development-orange)

Ein moderner, intuitiver Rentenrechner und Finanzplaner, der Licht ins Dunkel der Altersvorsorge bringt. Mit dem **Pension Planer v2** kannst du verschiedene Rentenszenarien simulieren, Rentenlücken identifizieren und deine finanzielle Zukunft strategisch planen.

---

## 🚀 Features

* **Partner-Modus:** Simuliere deine Pension zusammen mit deinem Lebenspartner
* **Schweizer Vorsorgewerk:** Berücksichtigung aller 3 Säulen, inklusive Steueroptimierer bei Kapitalbezug im Alter
* **Szenarien-Rechner:** Simuliere Früheinstieg, Regelaltersrente oder flexible Rentenmodelle.
* **Inflations- & Steuerberechnung:** Realistische Hochrechnungen durch Berücksichtigung von Kaufkraftverlust und voraussichtlicher Besteuerung.
* **Visuelle Dashboards:** Interaktive Graphen, die dir zeigen, wie sich dein Vermögen und deine Rentenansprüche über die Zeit entwickeln.
* **Zinseszins-Simulator:** Integriere private Vorsorge (ETFs, Aktien, Lebensversicherungen) und sehe den Hebeleffekt.

## 📋 Voraussetzungen & Installation

Damit das Projekt lokal läuft, folge diesen einfachen Schritten:

### 1. Repository klonen
```bash
git clone [https://github.com/Thagu/pension-planer-v2.git](https://github.com/Thagu/pension-planer-v2.git)
cd pension-planer-v2
2. Abhängigkeiten installieren
(Passe diesen Befehl an dein Projekt an, z.B. npm install oder pip install -r requirements.txt)

Bash
npm install
3. Anwendung starten
Bash
npm start
Öffne anschließend http://localhost:3000 in deinem Browser.
