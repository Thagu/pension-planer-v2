# Pensionsplaner Schweiz - Project Instructions

## Overview
Swiss pension planning PWA (Progressive Web App) for the Swiss market. Semi-professional precision (Stufe B), built with iterative time-series simulation (Pipe-Prinzip).

## Architecture Decisions
- **Profile vs. Scenario separation**: Personal data (birthDate, retirementAge, salary, BVG, 3a, freeAssets) lives on the Profile. Scenarios are "what-if" containers that inherit profile data and add overrides.
- **Engine portability**: `/lib/engine/` is UI-agnostic (no browser APIs) for future React Native migration.
- **Simulation**: Pipe-based projection engine (planned), client-side computation for low latency.
- **Multi-persona ready**: Architecture supports future multi-profile scenarios (one user, multiple personas).

## Data Model
- **Profile**: birthDate, gender, employmentStartYear, retirementAge, currentSalaryBrutto, bvgCurrentCapital, pillar3aCurrentCapital, freeAssets
- **Scenario**: name, data (JSONB for overrides like different retirement age, assumptions, career breaks)
- Career breaks (Erwerbsunterbrüche) already foreseen in types

## Tech Stack
- Next.js 14, TypeScript, Tailwind CSS, Prisma (PostgreSQL)
- NextAuth (email/password), PWA manifest
- Framer Motion for animations, Sonner for toasts
- CHF formatting: `de-CH` locale

## Design
- Navy-blue primary color scheme (HSL 215 50% 23%)
- DM Sans + Plus Jakarta Sans fonts
- Professional, clean Swiss financial app aesthetic
- Responsive: Desktop, Tablet, Mobile

## MVP Roadmap
1. ✅ Auth (Login/Signup)
2. ✅ Profile form (persönliche Daten + Vermögen)
3. ✅ Scenario CRUD (name + description, overrides later)
4. ✅ AHV-Modul (1. Säule) — Rentenberechnung mit Beitragsjahren, Rentenskala, Vorbezug/Aufschub
5. ✅ BVG-Modul (2. Säule) — Altersguthaben, Umwandlungssatz, Koordinationsabzug
   - Profil-Grundlagen: Zinssatz, Umwandlungssatz, Altersgutschriften (4 Buckets) direkt im Profil einstellbar
   - Szenario-Overrides können diese Profil-Defaults überschreiben
6. 🔲 Säule 3a Modul — Kapitalprojektion (Zins im Profil bereits erfassbar)
   - Profil: 3a-Guthaben + jährlicher Zinssatz bereits im Profil
   - Engine-Modul für Kapitalprojektion noch ausstehend
